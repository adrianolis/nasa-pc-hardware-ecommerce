# 🚀 NASA Store

> E-commerce de produtos gamer e acessórios tech — projeto de conclusão de curso

---

## 📖 Sobre o Projeto

A **NASA Store** é um site de e-commerce voltado para a venda de produtos gamer e acessórios de tecnologia. Desenvolvido como projeto de conclusão de curso, o site conta com uma interface moderna, navegação intuitiva e layout responsivo com Bootstrap.

🔗 **Acesse o site:** [seu-usuario.github.io/nasa](https://seu-usuario.github.io/nasa/)

---

## 📄 Páginas

| Página | Arquivo | Descrição |
|---|---|---|
| Página Inicial | `index.html` | Carousel de banners e informações gerais |
| Promoções | `promocoes.html` | Produtos com desconto |
| Mais Vendidos | `mais-vendidos.html` | Produtos mais vendidos com filtro por avaliação |
| Carrinho | `carrinho.html` | Carrinho de compras com cálculo de totais |
| Checkout | `checkout.html` | Página de pagamento e envio |
| Confirmação | `confirmação.html` | Confirmação de pedido |
| Login | `login.html` | Acesso à conta do cliente |
| Cadastro | `cadastro.html` | Criação de nova conta |
| Recuperação de Senha | `recuperacao.html` | Formulário para redefinir senha |
| Contatos | `contatos.html` | Informações de contato e redes sociais |
| Login Efetuado | `feito.html` | Página de confirmação de login |

## 🚀 Como Visualizar

### Online
Acesse diretamente pelo link do deploy:
**[seu-usuario.github.io/nasa](https://seu-usuario.github.io/nasa/)**

### Localmente
1. Clone o repositório:
   ```bash
   git clone https://github.com/seu-usuario/nasa.git
   ```
2. Abra o arquivo `index.html` diretamente no navegador.

> Não é necessário instalar dependências — o projeto usa apenas arquivos estáticos.

---

## ⚠️ Observações

As funcionalidades de login, cadastro e recuperação de senha requerem um servidor com PHP e banco de dados MySQL para funcionar completamente. No deploy estático (GitHub Pages), essas páginas são exibidas mas o processamento do formulário não é executado.
